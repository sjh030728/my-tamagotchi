# CLI Tamagotchi Simulator 

> **C++ 기반 순수 텍스트(CLI) 가상 펫 육성 시뮬레이션 게임**

## 📖 프로젝트 소개
터미널 환경에서 즐길 수 있는 클래식한 턴제 다마고치 시뮬레이터입니다. C++의 객체 지향 프로그래밍(OOP) 패러다임을 적용하여 개발되었으며, 윈도우(Windows) CMD 등 구형 터미널에서 발생하는 **문자 인코딩(깨짐) 문제를 원천 차단하기 위해 100% 순수 영문과 ASCII 기호만으로 설계**되어 완벽한 크로스 플랫폼 호환성을 자랑합니다.

## ✨ 주요 기능 및 특징
* **🎯 명확한 클리어 목표:** 무한 루프 생존 게임을 넘어, 펫을 **30턴(Age 30)까지 생존**시키면 성체로 성장하며 게임을 클리어하는 엔딩 시스템을 도입했습니다.
* **📺 순수 ASCII 동적 그래픽:** 펫의 현재 상태(행복, 배고픔, 피로, 사망 등)에 따라 실시간으로 변화하는 ASCII 아트 표정을 구현했습니다.
* **💾 세이브 & 로드 (데이터 영속성):** 언제든 게임을 중단하고 `tamagotchi_save.txt` 파일에 데이터를 안전하게 저장한 뒤, 다음 실행 시 이어서 플레이할 수 있습니다.
* **🛡️ 강력한 입력 예외 처리:** 사용자가 메뉴에서 숫자가 아닌 문자나 특수기호를 입력하더라도, 내부 버퍼를 즉시 비우고(`cin.fail()`, `cin.clear()`) 재입력을 요구하여 프로그램이 절대 다운되지 않습니다.

---

## 🚀 빌드 및 실행 방법

본 프로그램은 C++11 이상을 지원하는 모든 환경에서 구동됩니다. 사용하시는 개발 환경에 맞춰 아래 방법 중 하나를 선택해 주세요.

### 방법 A. IDE(통합 개발 환경) 사용자
Visual Studio, VS Code, CLion, Dev-C++ 등을 사용 중이시라면, 소스코드를 열고 **실행(Run / 단축키 F5 또는 Ctrl+F5)** 버튼을 누르시면 터미널이 열리며 게임이 바로 시작됩니다.

### 방법 B. 터미널(CLI) 직접 컴파일 사용자
IDE 없이 순수 터미널 환경에서 실행하시려면, 소스코드가 있는 경로에서 아래 명령어를 순서대로 입력해 주세요.

```bash
# 1. 컴파일 (Compile)
g++ -std=c++17 main.cpp -o tamagotchi

# 2. 실행 (Run)
# Mac / Linux 환경
./tamagotchi

# Windows 환경
tamagotchi
